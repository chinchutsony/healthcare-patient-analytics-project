# Healthcare Patient Analytics Project

## Project Overview
This project analyses healthcare patient data to identify trends related to hospital efficiency, treatment cost, patient recovery, and operational performance.

## Dataset
Healthcare Patient Analytics Dataset from Kaggle.

## Tools Used
- Microsoft Excel
- Jupiter Notebook
- SQL
- Power BI
- GitHub


## Day 1 Progress
- Loaded healthcare dataset
- Checked missing values
- Calculated average LOS
- Analysed treatment cost
- Identified busiest department

## Initial Findings
-Total records: 5000
- Average LOS: 4.06 days
- Avg Treatment Cost: 54915.47
- Highest Patient Department: Orthopedics
- Avg Recovery Score: 74.72
